# CivicLens Frontend (Next.js)

This directory contains a modern Next.js App Router frontend scaffold tailored for CivicLens.

Features:
- Next.js (App Router) + TypeScript
- TailwindCSS (dark mode by default)
- Recharts for charts
- Axios base integration to `http://127.0.0.1:8000`
- Responsive sidebar layout and top navbar

Quick start

1. Install dependencies

```bash
cd frontend_next
npm install
```

2. Run development server

```bash
npm run dev
```

Environment

- You can set `NEXT_PUBLIC_API_URL` to point to your backend if different from the default `http://127.0.0.1:8000`.

Notes

- This is a scaffold with placeholders for components and services. Replace placeholder content with production-ready UI and authentication flows.
